# ClinicQueueApp

ClinicQueueApp is a simple web application built using Flask that manages a clinic patient queue.

## Features
- Add patients
- View patients on the home page
- Serve (remove) the first patient in the queue
- Displays total number of patients

## How to run the app
1. Make sure Python is installed
2. Open the project folder
3. Open Command Prompt inside the folder
4. Install Flask if needed:
   pip install flask
5. Run the app:
   python app.py
6. Open your browser and go to:
   http://127.0.0.1:5000/
